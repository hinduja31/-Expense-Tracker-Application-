Expense Tracker GUI

Expense Tracker GUI is a Java Swing-based desktop application developed to manage daily expenses efficiently. The application provides features such as secure login authentication, expense management (add, update, delete, search, filter, and sort), dashboard summaries, category-wise and monthly reports, budget alerts, savings calculation, and data persistence using file handling. Built using Core Java and Swing, this project demonstrates object-oriented programming, GUI development, file management, and exception handling concepts in a practical real-world application.
